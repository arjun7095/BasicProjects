# To-Do List Application

## Project Overview
This **To-Do List Application** is a simple, web-based task management tool built with HTML, CSS, and JavaScript. It allows users to add, edit, delete, and mark tasks as completed. The application is fully responsive and retains task data using browser local storage, so tasks persist across page reloads.

### Features:
- Add new tasks.
- Edit existing tasks directly on the page.
- Mark tasks as completed or incomplete using checkboxes.
- Delete tasks with a confirmation prompt.
- Data is stored locally in the browser (Local Storage).
- Responsive design for mobile, tablet, and desktop views.
- Advanced CSS design with a modern look.

## Technologies Used:
- **HTML**: Structure of the application.
- **CSS**: Styling and layout with responsive design and modern UI.
- **JavaScript**: Functionality for task management, local storage, and interactivity.
